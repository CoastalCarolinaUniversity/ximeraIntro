\section{Ximera Environments}

Each of the following environments is created with the command $\backslash$begin\{name of environment\}.
Add .tex to the end of the url to see the source LaTeX code.

The 'theorem' environment:
\begin{theorem}(Increasing/Decreasing)
    Suppose $f$ is differentiable on the open interval, $I$. If $f'(x) >0$ on $I$ then $f$ is {\bf increasing} on $I$, and similarly
    if $f'(x) <0$ on $I$ then $f$ is {\bf decreasing} on $I$.
\end{theorem}

The 'proposition' environment:
\begin{proposition}
    Let $\mu$ and $\nu$ be eigenvalues of $A$...
\end{proposition}

The 'lemma' environment:
\begin{lemma}
    If $u \perp v$ and $u \perp w$, then...
\end{lemma}

The 'corollary' environment:
\begin{corollary}
    An eigenspace is at least one dimensional.
\end{corollary}

The 'remark' environment:
\begin{remark}
    The sum of the dimensions of the eigenspaces is...
\end{remark}

The 'example' environment:
\begin{example}
    Roses are red.
\end{example}
In addition to the normal tex output html format, ximera offers the answer box! The answer box is the most fundamental form of assessment feedback. It instantly lets a user know whether their answer was acceptable or not-and it has the patience to allow for as many corrections as are needed.

The 'problem' environment:
\begin{problem}
According to a result derived from the analytic continuation of the Riemann zeta function and Ramanujan summation, the sum of the natural the sum of the natural numbers is: $\answer{-\frac{1}{12}}$
\end{problem}

Explore the answer box!
Some of the special things the answer box can handle are: infty, pi, union, sqrt sin(x), ln(x) etc. Notice that as you type, an interpreter instantly converts your input to standard mathematical symbols. To get the green check mark, 
type -$\frac{1}{12}$ in the answer box :)

In addition to an answer box, Ximera provides multiple-choice, word-choice and select-all formats to significantly bolster the assessment feedback options.
\begin{problem}
Here is a multiple choice question.
Which of the following is an anti-derivative of $\ln x$?
    \begin{multipleChoice}
        \choice[correct]{$x\ln(x) - x$}
        \choice{$x\ln(x) - 1$}
        \choice{$x\ln(x) + x$}
    \end{multipleChoice}
    \begin{hint}
        Differentiate the options.
    \end{hint}
\end{problem}
Notice the hint- an excellent sub-environment to the problem environment.
Add .tex to the end of the url to see the source LaTeX code.

\begin{problem}
Here is a select-all question.
Select all of the correct statements:
    \begin{selectAll}
        \choice{If $f$ is continuous, then $f$ is differentiable.}
        \choice[correct]{If $f$ is differentiable then $f$ is continuous.}
        \choice[correct]{If $f$ is not continuous, then $f$ is not differentiable.}
    \end{selectAll}
\end{problem}

Here is a question with word choices:
\begin{question}
    In the 14th century, the French Philosopher and Mathematician Nicole Oresme famously demonstrated that the Harmonic Series \begin{wordChoice}{\choice{converges}\choice[correct]{diverges}}\end{wordChoice} by comparing it to an infinite series with constant terms.
\end{question}

\section{Embedding Interactives}

Ximera supports the following commands for easy embedding of output from other sources:
\text{\textbackslash youtube\{\}}: \youtube{AkYCqV75EOw}
\text{\textbackslash desmos\{\}}: \desmos{j9hdqgwiwg}{800}{600}
\text{\textbackslash geogebra\{\}}: \geogebra{vV7WGnrc}{1000}{750}\\
Embedding an image using the includegraphics command:
\begin{image}
    \includegraphics{missionPatch.jpg}
\end{image}
The above image is in the ``image" environment- $\backslash$begin\{image\}- and clicking on the image opens it up in a stand alone mode. (If you do click on the image, you can close it by clicking on the white 'X' in the top far-right corner.)

Ximera also supports embedding a format called doenet: \text{\textbackslash doenet\{\}}

\begin{proof}

\end{proof}
